📚 Online Bookstore 

 *📝 Description*  
The *Online Bookstore* is a web-based platform designed to provide users with a seamless experience for *browsing, searching, and purchasing books. It includes features like **user authentication, book catalog, shopping cart, order management, and secure checkout*.  

The project follows a *Scrum-based development approach* with roles such as *Product Owner, Developer, Tester (QA), Security Engineer, DevSecOps, and Deployment & Ops, ensuring a **secure, scalable, and efficient system*.  

## *🚀 Features*    
✅ *Book Catalog & Search* – Browse books by category, search, and filter results.  
✅ *Shopping Cart & Checkout* – Add books to the cart and complete purchases securely.  
✅ *Order Management* – Track past purchases and order details.  
✅ *Reviews & Recommendations* – Leave reviews and receive personalized book suggestions.  
✅ *Admin Dashboard* – Manage book inventory, track stock levels, and generate reports.  
✅ *Security Features* – Secure authentication, encryption, and compliance with *OWASP* best practices.  
✅ *Automated Deployment & Testing* – Using *Terraform, GitHub Actions, and Selenium* for CI/CD.  

## *🛠 Tech Stack*  
*Frontend*: Node.js  
*Backend*: Node.js,Express,Mongodb    
*DevOps & Security*: AWS, Terraform, GitHub Actions, OWASP Security Testing  
*Testing*: Selenium

## *👥 Team Roles & Responsibilities*  
- *Product Owner* – Defines features, prioritizes tasks.  
- *Developer* – Implements features like authentication, cart, and checkout.  
- *Tester (QA)* – Ensures system functionality with automated testing.  
- *Security Engineer* – Implements security measures (SAST/DAST).  
- *DevSecOps* – Manages CI/CD pipelines with GitHub Actions.  
- *Deployment & Ops* – Deploys the bookstore on AWS/Azure.
